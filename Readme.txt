This is the open-source code and data file for JESTIE paper 

Title: A Unified Framework for System-Level Condition Monitoring of Power Electronic Converters
Anuthor: Shuyu Ou, Student Member, IEEE, Subham Sahoo, Senior Member, IEEE, Ariya Sangwongwanich, Senior Member, IEEE, Frede Blaabjerg, Fellow, IEEE, Mahyar Hassanifar, Student Member, IEEE, Martin Votava, Member, IEEE, Marius Langwasser, Member, IEEE, and Marco Liserre, Fellow, IEEE.

File1: 
J3_optmization_comparison_20251108.ipynb
Target: Using the simulation data compare input signals and optimization algorithms.

File2:
J3_experimental_data.ipynb
Target: Process the experimental data to achieve accurate estimations.

File3: 
J3_boost.ipynb
Target: Validate the proposed framework can also be used in the boost converter parameter estimations with slight modifications. 
-------------------------------

Note: Remember to change the data path before running the code.
